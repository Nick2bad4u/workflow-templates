# Contributing to workflow-templates

Thanks for helping improve this workflow template repository.

## Local setup

```bash
npm install
npm run lint
```

## Development guidelines

- Keep workflows minimal, explicit, and easy to audit.
- Use least-privilege permissions in all workflows.
- Pin GitHub Action versions.
- Keep template examples practical and easy to customize.

## Pull requests

- Keep PRs focused and small.
- Include a short rationale in the PR description.
- Update README when adding or changing template workflows.
