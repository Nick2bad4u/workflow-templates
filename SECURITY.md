# Security Policy

## Reporting a Vulnerability

Please do **not** open public issues for security vulnerabilities.

Report privately using:

- GitHub Security Advisories: <https://github.com/Nick2bad4u/gh-runs-cleanup/security/advisories/new>
- Maintainer email: <20943337+Nick2bad4u@users.noreply.github.com>

Include reproduction steps, impact, and affected versions when possible.
